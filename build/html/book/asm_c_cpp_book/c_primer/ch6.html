第六章 C控制语句：循环¶

本章介绍以下内容：
  • 关键字：for、while、do while
  • 运算符： <、>、>=、<=、!=、==、+=、*=、-=、/=、%=、
  • 函数：fabs()
  • C语言有三种循环：for、 while 、do while
  • 使用关系运算符构建控制循环的表达式
  • 其他运算符
  • 循环常用的数组
  • 编写有返回值的函数
对于计算机科学而言，一门语言应该提供以下3种的程序流：
  • 执行语句序列
  • 如果满足某些条件就重复执行语句序列(循环)；
  • 通过测试选择执行哪一个语句序列(分支)
