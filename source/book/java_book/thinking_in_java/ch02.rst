垃圾回收机制：

java的垃圾回收机制用来件事用new创建的所有对象  并辨别那些不会再被引用的对象。

static关键字:

当一个十五是static时，就意味这个域或者方法不会被包含它的那个类的任意对象实例关联在一起。
