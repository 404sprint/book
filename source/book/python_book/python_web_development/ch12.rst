介绍了一些linux命令：

uptime：获取当前系统的平均负载

free：获取当前内存的使用情况

htop：运行于linux系统的监控与进程管理软件

dstat：是用python实现的多功能系统资源统计生成工具。

Glances：是基于psutil的跨平台的系统监控工具。

iftop：类似于top命令的实时流量监控工具。

sysdig：是linux服务器监控和故障排除工具

nethogs：查看进程暂用宽带情况

iptraf:网络流量监控工具。
